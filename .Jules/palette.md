## 2026-01-01 - Click-to-Copy UX Pattern
**Learning:** Developers appreciate when file paths in documentation are clickable to copy, reducing friction in setup. Providing immediate visual feedback ("Copied!" tooltip) is critical for this micro-interaction to feel responsive.
**Action:** When displaying file paths or command snippets in onboarding/documentation UIs, wrap them in a `CopyableCode` component with accessible feedback.
